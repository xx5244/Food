# 泡芙
---
+ ## 組成
  1. 奶油
  2. 液體
  3. 麵粉
  4. 調味

+ ## 20210808
  + ### 材料
    1. 奶油
    2. 鮮奶
    3. 高筋麵粉
    4. 砂糖
    5. 鹽
  
  + ### 作法
    1. 將鮮奶、奶油、砂糖、鹽蹈入鍋子中加熱拌勻
    2. 加熱到冒泡煮開關火，倒入過篩麵粉攪拌均勻
    3. 開小火拌炒至鍋底有白色黏稠
    4. 倒入容器放涼備用
    5. 打散雞蛋
    6. 先倒入一些雞蛋液至奶油糊攪拌觀察狀態
    7. 觀察到手指畫下去痕跡不會消失即可
    8. 將泡芙糊倒入擠花袋中
    9. 擠出至烘培紙上
    10. 烤箱預熱至190度左右
    11. 烤33分鐘
